欢迎使用vue-uweb插件

1. 安装

npm

npm install vue-uweb --save

直接在页面中引用

<script src="../node_modules/vue-uweb/dist/index.js"></script>

通过es6模块加载

import UWeb from 'vue-uweb'

使用vue-uweb

Vue.use(UWeb,'YOUR_SITEID_HERE')

2. uweb directives

track-event

<button v-track-event.click="'click, event''"></button>
<button v-track-event="'shortcut, event'"></button>
<input v-track-event.keypress="'keypress, event'">
关于参数和顺序 
<button v-track-event="'click, event'"></button>
<button v-track-event="{category:'click', action:'event'}"></button>

track-pageview

指定受访页面
<div v-track-pageview="'/bar'"></div>
以字符串指定受访页面和来源
<div v-track-pageview="'/tar, https://github.com/raychenfj'"></div>
以对象字面量指定受访页面和来源
<div v-track-pageview="{content_url:'/zoo', referer_url:'https://github.com/raychenfj'}"></div>
通过a标签指定受访页面 
<a href="/foo" v-track-pageview></a>

auto-pageview

autoPageView: {{auto}}
启用 auto-pageview
<div v-auto-pageview=true></div>
停止 auto-pageview
<div v-auto-pageview=false></div>
